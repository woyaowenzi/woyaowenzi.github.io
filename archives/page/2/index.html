Mac OS Tip

Q: Mac下Vim编辑器最简单语法高亮配置方法
A:

  1. 拷贝默认的配置文件: cp /usr/share/vim/vimrc ~/.vimrc
  2. 编辑该文件: vim ~/.vimrc
  3. 在文件的最后加入: syntax on
  4. 保存退出即可
